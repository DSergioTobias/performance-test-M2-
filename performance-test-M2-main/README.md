## Project title.
-Page for a Shattered Riffs band

## Brief description of the project.
-This page is for a hard rock band to show every information about:
*A new upcoming tour 
*Upcoming events 
*About the band
*Show social medias

## Technologies used.
-(HTML5, CSS3)

## Project structure.
-This project structure is based on:
*semantic tags:

<header>
<nav>
<main>
<section>
<article>
<footer>

*Include the following sections:

Hero (main visual section)
Events (band tour dates)
About (band info)
Multimedia (images or media content)

*Add at least one unordered list (<ul>):

Influences
Values
Social links

*Use a table (<table>) for events

*I Define my own:

Colors (rock style recommended)
Typography
Spacing

*Apply layout techniques:

Flexbox
Grid

*Implement responsive design using media queries

*Ensure:

Proper spacing between sections
Visual hierarchy 
Consistent styling

## How to run or view the project locally.
1.Clone or download this repository to your computer.
2.Open the folder.
3.Double-click index.html to open it directly in any modern web browser (Chrome, Firefox, Edge, Safari).
4.No build tools, servers, or dependencies required — it runs entirely in the browser.
Tip: For the best experience and to ensure Google Fonts load correctly, open the file with an internet connection, or use a local server such as VS Code's Live Server extension.
